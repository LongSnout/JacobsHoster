# Jacob’s Hoster

Aplicación de escritorio para la gestión de albergues del Camino de Santiago.

Este proyecto está desarrollado como aplicación local orientada a facilitar el registro, control y administración de peregrinos en un entorno sencillo y funcional.

---

## 🧭 Objetivo

Jacob’s Hoster está pensado como una solución ligera para:

- Gestión de peregrinos
- Registro de estancias
- Generación de informes XML
- Organización de habitaciones y camas
- Control básico administrativo

La aplicación está diseñada para funcionar de manera local sin necesidad de conexión permanente a internet.

---

## 🛠 Tecnologías utilizadas

- Java
- JavaFX
- SQLite
- JDBC
- Eclipse IDE

---

## ☁️ Servicios adicionales

Este repositorio contiene únicamente la versión local de la aplicación.

Los servicios en la nube, sincronización remota u otras funcionalidades avanzadas no forman parte de este repositorio y pueden ofrecerse como servicios independientes.
